# Flask Blog App

A blog application with authentication and CRUD operations.

## Features
- User registration
- User login
- Secure password hashing
- Add blog posts
- Delete blog posts
- Session authentication
- SQLite database

## Technologies Used
- Python
- Flask
- SQLite3
- bcrypt
- HTML

## Installation

pip install flask bcrypt

## Run

python app.py

Open browser:
http://127.0.0.1:5000

## Purpose
focuses on building a mini full-stack style Flask application.
